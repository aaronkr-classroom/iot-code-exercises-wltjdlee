# chp2 연습문제


2. 마이크로파이썬의 펌웨어를 올리기 위해서 부팅할 때 별도의 버튼을 클릭하게 합니다. 해당 절차의 필요성을 설명하세요.
마이크로파이썬 펌웨어를 보드에 업로드할 때 부팅 시 특정 버튼을 누르는 절차는 장치를 일반적인 프로그램 실행 모드가 아닌 펌웨어 업데이트 모드로 강제 진입시키기 위한 하드웨어적 제어 과정입니다.
기본적으로 마이크로컨트롤러는 전원이 켜지면 내장된 플래시 메모리에 저장된 기존 프로그램을 즉시 실행하도록 설계되어 있으나, 새로운 펌웨어를 기록하기 위해서는 프로세서가 일반 실행 경로를 멈추고 외부로부터 전송되는 데이터를 받아들여 메모리에 쓸 준비를 해야 합니다.
 이때 특정 버튼을 누른 채 부팅하면 특정 GPIO 핀의 논리 레벨이 변경되며, MCU 내부에 하드코딩된 ROM 부트로더가 이를 감지하여 통신 인터페이스를 통해 데이터를 수신할 수 있는 상태로 전환됩니다. 
이러한 절차는 소프트웨어적인 명령만으로는 제어가 불가능한 시스템 오류나 무한 루프 상황에서도 하드웨어 수준에서 강제로 새 펌웨어를 덮어쓸 수 있도록 보장하는 최소한의 안전장치이며, 사용자 프로그램의 의도치 않은 펌웨어 오염을 방지하고 장치의 복구 가능성을 확보하기 위한 핵심적인 통제 단계라고 할 수 있습니다.
